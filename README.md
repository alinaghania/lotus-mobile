# EDM Health App

## Installation

```bash
npm install
```

## Lancement en développement

```bash
npm run dev
```

## Build de production

```bash
npm run build
```

## Aperçu de production

```bash
npm run preview
```

---

- Nécessite Node.js 16+ et npm.
- Utilise React, Vite, TypeScript, Tailwind CSS, Lucide-react.
